# ربات رزرو مشاوره تحصیلی

این پروژه برای استقرار مستقیم روی GitHub و Render آماده شده است.

## امکانات

- دکمه «رزرو مشاوره تحصیلی»
- دریافت نام و نام خانوادگی
- دریافت شماره تماس با دکمه اشتراک‌گذاری یا ورود دستی
- دریافت شهر
- نمایش خلاصه و تأیید نهایی
- ارسال درخواست کامل برای ادمین
- فرمان `/id` برای پیدا کردن شناسه عددی ادمین
- اجرای Webhook روی Render

## راه‌اندازی

1. همه فایل‌ها را در یک Repository عمومی یا خصوصی GitHub آپلود کنید.
2. در Render گزینه **New > Blueprint** را بزنید.
3. Repository را انتخاب کنید.
4. هنگام ساخت سرویس، مقدار این متغیرها را وارد کنید:
   - `BOT_TOKEN`: توکن BotFather
   - `ADMIN_CHAT_IDS`: شناسه عددی ادمین
5. Deploy را اجرا کنید.
6. در تلگرام به ربات پیام `/start` بدهید.

## پیدا کردن ADMIN_CHAT_IDS

ابتدا ربات را Deploy کنید و در تلگرام فرمان `/id` بفرستید. عددی که ربات می‌دهد را در Render، بخش Environment، به‌عنوان `ADMIN_CHAT_IDS` ذخیره کنید و سپس Manual Deploy بزنید.

چند ادمین را می‌توان با ویرگول وارد کرد:

`123456789,987654321`

## امنیت

توکن را داخل GitHub، فایل کد، README یا اسکرین‌شات عمومی قرار ندهید. توکن فقط باید در Environment Variables سرویس Render ذخیره شود.
